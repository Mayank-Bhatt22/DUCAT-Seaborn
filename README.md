## Description
This project focuses on data visualization using the Seaborn library in Python. It includes various plots such as line plots, bar plots, histograms, box plots, and heatmaps with built-in styling for clear and effective data analysis.

## Tools & Libraries
- Python
- Seaborn
- Matplotlib
- Pandas
- NumPy

## Visualizations Covered
- Line Plot (sns.lineplot)
- Bar Plot (sns.barplot)
- Histogram (sns.histplot)
- Box Plot (sns.boxplot)
- Heatmap (sns.heatmap)

## Objective
To understand statistical data visualization and present insights in a visually appealing manner.

## Learning Outcome
- Create attractive plots easily
- Analyze data using statistical graphics
- Improve data visualization skills
